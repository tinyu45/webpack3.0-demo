## 手动编译
```shell
 npm run webpack ./src/index.js bundle.js
```
## 自动编译
依赖 <span style="background:yellow">webpack.config.js</span>文件配置
```shell
npm run webpack
```

配置dev-server启动
``` shell
npm run start
```
## 注意
依赖版本尽量保持一致，好多坑😭
